# english_vocablary_test

メモ帳にカンマ区切りで書いた単語を四択問題として出す。
英語->日本語モードと日本語から英語モードが選択できる。

eng.txtとしてcppファイルと同じ場所に保存する。
```
perch,止まり木
pile,積み重ね
clutter,散らかっているもの
remarkeble,注目すべき
...
```

モード選択をしたらが演習がスタートする。

各問題ごとに
5秒以内で正答できた時, +1
正答できたが5秒以上かかったとき, -1
正答できなかったとき, -2

で評価わけされ、合計が10以上のときに演習を終了とする。

コンパイルはしていません。
